Plate Quest v1.2 visual refresh

Upload the included app folder to the root of your existing PlateQuest GitHub repository.
Commit the changes, wait for the GitHub Actions build to finish, then download the new APK artifact.

What changed:
- New gradient header and progress hero
- License-plate styled entry field
- More polished cards, badges, spacing, and typography
- Seven visual position tiles with individual completion rates
- New next-targets card showing missing letters and numbers
- Improved collection grid and position selector
- Redesigned recent-plate history
- Updated app theme and version 1.2.0

Saved game progress uses the same preference keys, so installing over the existing app should preserve progress when Android permits the update.
